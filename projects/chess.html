Online Chess Studio

A real-time chess platform focused on smooth gameplay, analysis tools, and clean UI.

TypeScript Node.js WebSockets

This project explores real-time multiplayer infrastructure with a focus on latency and reliability. The client side prioritizes readable board states, quick move confirmation, and game review.

I built the matchmaking service, state sync, and a minimal analysis board to revisit key positions.

View on GitHub

Technical challenges

Implementing reliable game state sync and reconnection logic while keeping payloads small. The experience taught me how important deterministic state and server authority are for real-time apps.

What I learned

Designing a protocol that balances speed and clarity, and building UI states that handle errors gracefully without interrupting gameplay.
