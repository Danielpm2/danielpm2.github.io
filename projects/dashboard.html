City Data Dashboard

Interactive dashboard that translates public APIs into actionable local insights.

JavaScript REST APIs Data Viz

This dashboard combines environmental, transit, and event data into a single responsive view. The goal is to make civic data feel immediate and easy to scan.

I designed a modular data-fetching layer with caching to keep API calls predictable and to enable fast client-side filtering.

View on GitHub

Technical challenges

Managing inconsistent API schemas and rate limits. I built a normalization step and added fallbacks so the UI remains useful even when data is incomplete.

What I learned

Strong data stories rely on responsible visual hierarchy. I practiced distilling complex datasets into small, focused visual cards.
