Mandelbrot Visualizer

A real-time fractal explorer built to understand GPU pipelines and numerical stability.

C++ OpenGL GLSL

The visualizer supports deep zoom, palette editing, and smooth navigation with a focus on keeping frames stable even at high magnification levels.

I implemented a custom shader pipeline, dynamic color maps, and a profiling overlay to tune GPU workloads.

View on GitHub

Technical challenges

Achieving stable zoom at extreme magnifications while preventing floating-point drift. I learned how to keep precision consistent between CPU calculations and fragment shaders.

What I learned

Building GPU workflows from scratch and profiling each stage taught me how to reason about bottlenecks and memory bandwidth in graphics programming.
