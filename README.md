# danielpm2.github.io
Clean, static portfolio website showcasing projects, graphics experiments, and systems work. Built with HTML, CSS, and vanilla JavaScript for GitHub Pages.

## Structure
- `index.html` — Home / hero overview
- `about.html` — About me narrative
- `projects.html` — Project grid
- `contact.html` — Contact links and static form placeholder
- `projects/` — Individual project case-study pages
- `css/style.css` — Global styling
- `js/main.js` — Navigation + subtle interactions
- `assets/images/` — Placeholder images and visuals

## Run locally
Open any HTML file in your browser (for example, `index.html`). No build step required.

## Customize
- Update the name, links, and contact info directly in the HTML pages.
- Replace placeholder SVGs in `assets/images/` with real screenshots when available.
